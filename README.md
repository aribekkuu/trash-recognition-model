# TrashDetect Model
**TrashDetect Model** is a computer vision project designed to automatically detect and classify different types of trash using deep learning.
The goal is to support smarter waste sorting systems and promote environmental sustainability through AI.
**Dataset** and **Labels** are from **Kaggle**.

## Tech
**Languages:** Python 3.14
**Frameworks:** Tensorflow, Keras, PyTorch
**Libraries:** NumPy, Pandas, OpenCV, Pillow, Seaborn

## Quick Start
### In Google Colab or Jupyter
- Download `trashdetect_model-2.ipynb` and dataset
- Upload and Run in platforms
